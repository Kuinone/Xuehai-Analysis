# Xuehai-Analysis


## Introduction
**Xuehai-Analysis** 是由破碎数据研究所（BDRG）基于先前项目研究成果的的针对XH平板及其附属应用、XH服务器的开源研究项目。

## Todo
目前项目主要研究方向有：
- Code校验及SecretKey获取
- ZTY及ZTP代码反编译
- ZTP-API的深度解析利用
- XH后端服务器探索
